# Parkinson's Disease Detection Chatbot

## Overview
This project implements a chatbot that utilizes voice frequency analysis to predict the likelihood of Parkinson's disease (PD) detection. The chatbot is built using Streamlit for deployment and leverages various data science libraries for analysis and visualization.

## Tech Stack
- **Google GenAI**: For natural language processing and generating responses.
- **dotenv**: For managing environment variables.
- **Matplotlib**: For static data visualizations.
- **Pandas**: For data manipulation and analysis.
- **Scikit-learn**: For machine learning model implementation.
- **Seaborn**: For enhanced data visualization.
- **Plotly**: For interactive graphs and plots.

## Features
- User-friendly chatbot interface for PD detection inquiries.
- Voice frequency analysis to assess PD risk.
- Interactive visualizations for data insights.
- Easy integration and deployment via Streamlit.
